CNHD Search Module
======================

Synology NAS/Diskstation Search Module For CNHD.COM/HD.GG

### 对于使用nexusphp搭建的PT站都适用，只需更改search.php中的主机名即可

### 安装
将bin文件夹内的search.dlm通过DownloadStation进行安装
